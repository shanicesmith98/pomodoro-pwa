# Pomodoro Focus — PWA

A dark-mode pomodoro timer with a built-in task list, designed for focus. Installable on your iPhone Home Screen as a native-feeling app.

## Features

- **Three modes** — Focus (25 min), Short Break (5 min), Long Break (15 min), all customizable
- **Auto-cycling** — automatically suggests a long break every 4 sessions
- **Task list** — add, edit (double-tap), complete, and delete tasks
- **Persistent** — tasks, settings, and session count survive page refreshes (localStorage)
- **Notifications** — browser push notification when a timer completes
- **Wake Lock** — screen stays on during active sessions (where supported)
- **Offline-ready** — service worker caches all assets for full offline use
- **Installable** — meets PWA criteria for Add to Home Screen on iOS and Android

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at `http://localhost:5173`. Hot-reloads on save.

## Build

```bash
npm run build
```

Output goes to `dist/`. The PWA manifest and service worker are auto-generated by `vite-plugin-pwa`.

## Preview the production build locally

```bash
npm run preview
```

## Deploy

Any static host works. A few free options:

### Vercel (recommended — zero config)
```bash
npm i -g vercel
vercel
```

### Netlify
```bash
npm i -g netlify-cli
netlify deploy --prod --dir=dist
```

### GitHub Pages
Push `dist/` to a `gh-pages` branch, or use the `gh-pages` npm package:
```bash
npm i -g gh-pages
gh-pages -d dist
```

### Cloudflare Pages
Connect your repo at https://pages.cloudflare.com — set build command to `npm run build` and output directory to `dist`.

## Install on iPhone

1. Deploy to any HTTPS host (required for service workers)
2. Open the URL in Safari
3. Tap the Share button → "Add to Home Screen"
4. The app launches full-screen with its own icon — no browser chrome

## Customization

- **Colors / modes** — edit the `MODES` object in `src/App.vue`
- **Fonts** — swap the Google Fonts import in `index.html` and `main.css`
- **Icons** — replace the PNGs in `public/` (192×192, 512×512, and 180×180 apple-touch-icon)

## Tech stack

- Vue 3 (Composition API)
- Vite 5
- Tailwind CSS 3
- vite-plugin-pwa (Workbox service worker)
