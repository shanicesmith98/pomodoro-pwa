# Pomodoro Focus — PWA

A pomodoro timer with task management, ambient sound, and a progression system. Dark-mode, installable as a native-feeling app on iOS and Android.

## Features

### Timer
- **Three modes** — Focus (25 min), Short Break (5 min), Long Break (15 min), all customizable
- **Auto-cycling** — switches to short break after each session; long break every 4 sessions
- **Session completion overlay** — satisfying animation and chime when a focus block ends
- **Wake Lock** — screen stays on during active sessions (where supported)
- **Notifications** — browser push notification on timer completion

### Sound
- **Ambient audio** — rain sounds during focus, lo-fi tones during breaks (Web Audio API, no files)
- **YouTube music** — paste any YouTube video or playlist URL in settings; separate URLs for focus and break modes; syncs play/pause with the timer automatically
- **Session chimes** — ascending arpeggio with chord bloom on work complete; gentle descend on break end
- **Toggle** — ambient sound on/off in settings; YouTube player shows inline when a URL is set

### Tasks
- **Task list** — add, edit (double-tap), complete, and delete tasks
- **Focus card** — surfaces your top pending task when a work session is running
- **Pomodoro estimates** — set an estimated session count per task inline
- **Calibration tracking** — completed tasks show estimated vs actual pomodoro count with a color-coded accuracy label

### Time awareness
- **Today's focus bar** — passive progress bar showing minutes focused vs a daily target (4 sessions)
- **Animated ring track** — the timer ring track slowly shifts color as a session progresses
- **Break protection** — task list locks during breaks with a stretch prompt; break-ending warning at 80% elapsed

### Progression
- **XP system** — earn 10 XP per completed work session
- **10 levels** — Seedling → Sprout → Sapling → Focused → Consistent → Dedicated → Unstoppable → Flow State → Deep Work → Legendary
- **Level badge** — shows current level, title, and XP progress bar between session dots and today bar

### Accessibility
- Full keyboard navigation (arrow keys on mode tabs)
- ARIA roles, labels, and live regions throughout
- WCAG 2.1 AA contrast on all text

### PWA
- **Offline-ready** — Workbox service worker caches all assets
- **Installable** — meets PWA criteria for Add to Home Screen on iOS and Android
- **Persistent** — tasks, settings, session count, XP, and YouTube URLs survive refreshes (localStorage)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at `http://localhost:5173`. Hot-reloads on save.

## Build

```bash
npm run build
```

Output goes to `dist/`. The PWA manifest and service worker are generated by `vite-plugin-pwa`.

## Tests

```bash
npm test          # single run
npm run test:watch  # watch mode
```

72 unit tests covering `useTimer`, `useTodos`, `useXP`, and `useAudio` composables.

## Preview the production build locally

```bash
npm run preview
```

## Deploy

The repo includes a GitHub Actions workflow that auto-deploys to GitHub Pages on every push to `main`. No manual steps needed once the repository's Pages setting is pointed at GitHub Actions.

For other hosts:

### Vercel
```bash
npm i -g vercel
vercel
```

### Netlify
```bash
npm i -g netlify-cli
netlify deploy --prod --dir=dist
```

### Cloudflare Pages
Connect your repo at https://pages.cloudflare.com — set build command to `npm run build` and output directory to `dist`.

## Install on iPhone

1. Open the deployed URL in Safari
2. Tap the Share button → "Add to Home Screen"
3. The app launches full-screen with its own icon

## Customization

| What | Where |
|---|---|
| Mode colors and labels | `src/config/modes.js` — `MODES` object |
| Stretch prompts | `src/config/modes.js` — `STRETCH_PROMPTS` array |
| XP per session and level thresholds | `src/composables/useXP.js` |
| Timer defaults (durations) | `src/composables/useTimer.js` — `loadJSON('pomo-durations', ...)` |
| Fonts | Google Fonts import in `index.html` |
| PWA icons | `public/` — replace `pwa-192x192.png`, `pwa-512x512.png`, `apple-touch-icon.png` |

## Project structure

```
src/
  components/
    BreakScreen.vue          — stretch prompt and break-ending warning
    ProgressRing.vue         — SVG timer ring
    SessionCompleteOverlay.vue — completion animation and XP display
    SettingsPanel.vue        — durations, ambient toggle, YouTube URLs
    TodoList.vue             — task list with focus card and calibration
    YouTubePlayer.vue        — embedded YouTube iframe player
  composables/
    useAudio.js              — Web Audio ambient engine, chimes, YouTube guard
    useTimer.js              — timer state machine, session logic, today tracking
    useTodos.js              — task CRUD, estimates, calibration
    useXP.js                 — XP and level system
    useYouTube.js            — URL parsing, IFrame API loader
  config/
    modes.js                 — MODES color config and STRETCH_PROMPTS
  tests/
    setup.js                 — Web Audio and browser API stubs
    useAudio.test.js
    useTimer.test.js
    useTodos.test.js
    useXP.test.js
```
